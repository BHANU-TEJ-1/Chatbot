# Gemini Streaming Chatbot (Python)

A simple **real-time streaming chatbot** built using **Google Gemini (Generative AI) Python SDK**.  
This project demonstrates how to:

- Load API keys securely using `.env`
- Initialize the Gemini client
- Create a persistent chat session
- Stream responses token-by-token in real time
- Handle API and runtime errors gracefully

---

##  Features

-  Real-time **streaming responses**
-  Secure API key loading using **dotenv**
-  Persistent chat session with Gemini
-  Clean CLI-based chatbot interface
-  Robust error handling
-  Lightweight and fast execution

---

##  Tech Stack

- **Python 3.10+**
- **Google Gemini SDK**
- **python-dotenv**
